## Features
- **Anti-Freeze System:** Uses direct HTTP requests instead of heavy API calls to prevent "Loading" loops.
- **Speed:** Fetches data in milliseconds.
- **No Login Required:** Works by scraping public meta tags.
- **Smart Parsing:** Extracts Followers, Following, and Posts from the page metadata.
- **HD Avatar:** Retrieves the accessible profile picture.
- **Database:** Saves user history.
